Turbowarp Packager (how Meower gets it's .html files) has a feature to allow cloud variables that are named a certain way (ex. ☁ username) to behave differently than normal cloud variables. For Example, naming a cloud variable “☁ open link” allows it to open a url in a new tab when the value is set to that variable.

(Although, it would be hard to be able to register a user's click to a piece of text, so there could instead be a button on the bottom of a post which sends the user to that link.)
