Mini update: Meower will now hash passwords with SHA256 when sending an authentication request to CloudAccount. CloudAccount internally stores passwords hashed with SHA256, so now there's 2 times the hashing!

A future update I'd like to consider is salting hashes as well as adding more garbage data to ensure security.

I'm also working on a new project with @windowscj to improve the security of CloudLink, and we collectively call it CLTLS, or CloudLink TLS, and as the name implies, we're going to replicate the functionality of TLS in CloudLink. Hopefully, in the future, I'd like to implement it in Meower.
