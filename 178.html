I unofficially made some post interaction buttons for Meower+.



The HTML (not the same one used in the GIF because emojis break forum posts):
<span class="float-right">
<button class="postbutton white">1</button><button class="postbutton white">2</button>
</span>
(put this after the post title in your post header)

The CSS styling:
.float-right 
  {float: right;} 
.postbutton 
  {height:23px; position: relative; top:-3px; border-color: white; background-color: orange; padding: 4px; line-height: 0px; box-shadow: none; margin:0px 3px;}
.white
  {color: white;}
.postbutton:active 
  {background-color: #BB7A00;}

Working post hiding button:



The HTML:
<button onclick="removePost(this);" class="postbutton white">X</button>
(Juist insert this in the post buttons span.)

The JavaScript:
function removePost(button) {
	let x = button.parentElement.parentElement.parentElement; 
	let s = x.previousSibling;
	if (s.tagName === "BR") {
		s.remove();
	}
	x.remove();
}
