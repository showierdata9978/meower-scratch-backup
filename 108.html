BlockPoint wrote:

In a later update of Meower, can you add modding capabilities? Not cheats. Just a tool box that allows scratchers to customize the program and add small features

If I recall earlier, we had another sister project, Woofer, which is similar to that?

I made the project versatile enough to allow custom theme creation, it's very limited for customization features. A lot of the functionality in Meower is hard-coded and would require a major overhaul to add said features. There's a reason why I removed the “Automatic Translation”, “Auto TTS”, etc. features since they will take longer to implement.

Our current priority is to figure out a phrase selector GUI and add it. From there we do a bit of connecting code and final testing.

In the future, if you really wanted to play around with modding, I would suggest forking Yoom (Meower's feature-rich, yet forbidden, sister project).
