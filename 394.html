Hello everyone, so I’m not sure if this was accidentally overlooked on GitHub, or ignored, but I thought this might be useful for the Meower project, considering we are moving away from scratch.

What I've done is finally finished my chat system I’ve talked so much about! It includes:

Over 500 prebuilt phrases
1000s of words that can be typed, which have been whitelisted and are complete.
And most importantly, can easily be implemented into something bigger.
Here is a link to the project: https://turbowarp.org/604793013
It's a complete reworking of BlueList, however it includes prebuilt phrases as well as a finished library of words. You'll find comments on every script explaining how it works. I imagine we could also add a way to put messages “under review”, but that would be up to Mike, whether or not you decide to use my program.

One thing to note is that you can't just type something at the moment. I've stripped the whole thing down to simplify it for others. It just works by changing the INPUT variable to whatever you would like to be “typing”.

I realize it looks pretty simple, but I didn’t want to make things overly complicated.
