.postbutton:active

Working post hiding button:



The HTML:
<button onclick="removePost(this);" class="postbutton white">X</button>
(Juist insert this in the post buttons span.)

The JavaScript:
function removePost(button) {
	let x = button.parentElement.parentElement.parentElement; 
	let s = x.previousSibling;
	if (s.tagName === "BR") {
		s.remove();
	}
	x.remove();
}
