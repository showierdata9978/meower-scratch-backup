*I think that selections of theme mode and style (when starting meower) should both be on the same screen (because now if people want to see how all theme possibilities look they need to choose the theme itself and then can click light and dark to see this theme's light and dark variants, but to see other theme's dark variant they would need to refresh website to choose that different theme and then they would get to choose to see its dark theme and if they wouldn't like it as much as the one before they would need to refresh website again) of course it isn't that bad because you can also change the theme's style and mode in the options menu, but first impressions are very important and I think that this change would improve them (very slightly, but still).
