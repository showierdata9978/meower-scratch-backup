Small update:

1. I wrote a bot that takes advantage of GitHub and Ngrok, so now I have a way to ab-“use” GitHub pages. Every time I run my bot, it opens a ngrok tunnel and updates a GitHub page with the most up-to-date ngrok IP, which can be fetched by a project. Pretty cool! This feature will be called “Dynamic CloudLink URLs”.
